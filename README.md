# Founding Titan Eren Yeager (Attack on Titan)

## 🦴 About This Project

This project is a creative visualization of **Eren Yeager's Founding Titan form** from the anime and manga series **Attack on Titan (Shingeki no Kyojin)**.

The artwork recreates the iconic skeletal appearance of the Founding Titan using procedural lines and geometric structures, highlighting its massive spine, elongated body, and distinctive skeletal anatomy.

## ✨ Features

- Procedural skeletal animation
- Minimalist black-and-white aesthetic
- Inspired by Eren Yeager's Founding Titan design
- Smooth line-based rendering
- Lightweight and customizable

## 🎬 Inspiration

The Founding Titan is one of the most recognizable forms in *Attack on Titan*, representing Eren Yeager's ultimate transformation during the final arc of the series. Its enormous skeletal structure symbolizes the devastating power of the Founding Titan.

## 📷 Preview

The visualization depicts:

- Extended skeletal spine
- Rib-like body structure
- Titan-inspired anatomy
- Minimal artistic representation
